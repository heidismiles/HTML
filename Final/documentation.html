General Notes

My project represents a web site for a (fictional) elementary school teacher at the beginning of a school year. As someone with a background in education, this idea interested me, and allowed me to use lots of fun colors and designs. Included pages are "Home," "Blog," "Calendar," "Lessons & Homework," and "About Me."

Page Size

Each page body element uses the property max-width: 1280px.

Main Headers

For each page, I have a main header, which contains a background image, a text box with a translucent background, and a horizontal rule. The header sizes were chosen based on each background image used. Each background image uses the background-size:cover property so that it fills the size of the background, and is responsive to window size.

In some pages (Blog and Lessons), the header text box has been rotated using the transform property.

Navigation

Each page has a horizontal navigation bar. This is a ul element, with each li element set to float left.

Layout

Page layout generally occupies 95% width. Article sections use width: 70%, and sidebars use width: 24%. Sidebars generally contain the teacher's (fictional) contact information, and some other items relevant to the page.

Style Sheets

All pages use the "torres-styles.css" style sheet. Inline / embedded styles were not used. The style sheet is well organized and commented, and the code is thoughtful and efficient.

IDs and classes

Each page's body element has a unique ID. These IDs are used for styles that apply only to one page. The IDs match the file names, e.g. #home, #blog, etc.

Layout elements have IDs, such as #mainHeader for the main header of the page. Certain style choices, such as grey backgrounds, were created using classes, such as .bg-lt (light background) and .bg-dk (dark background). I used classes for some styles that appeared repeatedly but arbitrarily across different pages and/or elements.

Fonts

The body text font is "Lato" and the header text is "ArchitectsDaughter". Fonts are stored with the page.

The elements strong and em were used to set bold and italic text, respectively, and these styles are explicitly set in the CSS. (Example: see the bulleted list on the Home page.) I also created the classes .strong and .em for the same purpose, but I don't believe I ended up using them. In "lessons.html", the dt elements (the course subjects) are set with font-variant: small-caps.

Colors and Backgrounds

Generally, the text is set to a shade of black (hsl(0, 0%, 25%) or hsl(0, 0%, 5%)). In the cases of dark or colored backgrounds (e.g. main page headers, footers), text is a shade of white (e.g. hsl(0, 0%, 100%) or hsl(0, 0%, 85%)). The classes .dark, .bg-lt, and .bg-md were also used to set text colors. An example of the .bg-lt class is in the sidebar of each page (with the text "* This contact information is fictional.")

Links are styled for text color, for link, visited, hover, and active states.

Each main header features a different background image, set with CSS. Page titles were set in a colored text box. These boxes used different hsla color values, with white font. Additionally, headers had a colored horizontal rule. Most pages used an aqua colored rule, and "lessons.html" used pink.

The navigation bar has a light blue background, with a darker blue on hover, as well as colored link states.

Tables (seen in "home.html" and "calendar.html" have colored headers.

The ::selection property was used to set the background color of selected text (a light blue). This might be extra credit, as I do not think this was covered in class.

Tables

The "home.html" page features a table, detailing a weekly class schedule. Borders are styled, with the outer border being thicker than the inner cell borders. The header row and left column are both set with a background color and bold text. Styles are set for padding, text-align, and vertical-align. Each column is set with width: 11%. The colspan attribute is used for some cells.

The "calendar.html" page features an even more verbose table, showing a monthly calendar for the academic year. The overall calendar is set in a 4 x 3 table. Each month is a nested table, occupying one of the 12 cells in the greater table. Borders are styled, with the outer borders being thicker than the inner cell borders. The header rows (for months and weekdays) are set with a background color and bold text. The weekday rows are set using the :first-of-type selector. Styles are set for padding, text-align, and vertical-align. Each column is set with width: 11%. The colspan attribute is used for some cells.

Additionally, for "calendar.html" I used classes to distinguish certain days on the calendar. The .bg-lt class was used for weekends to give them a grey background. For days off from school, I used the .day class to set a red background. And for the first/last day of the term, I used .term to set an aqua background.

Lists

The "home.html" page has a bulleted list, with some general notes about what to find on each page. Styles are set for padding, margin, list-style-position:outside, and list-style-type:disc.

The "blog.html" page has an unordered list in the sidebar. Bullets are removed, and padding is reduced for sidebar lists. Additionally, this page features the list-style-image property, setting the custom bullet ("time-icon.png") for the dates and times of events (see: "Back to School Night" and "Field Trip.")

The "calendar.html" page has a lengthy dl list. Classes are used to set background colors for school holidays, as well as the first/last day of each term. Styles are set for margin and padding, and for the font-weight of dt elements (the dates in the list).

The "lessons.html" page also has dl lists. Here, the dt elements (the course subjects) are set with font-variant: small-caps. Also, within each dt element is a ul element, showing multiple lessons for each subject.

 

Other Items

Gradient

The main navigation bar for each page uses a color gradient (background: linear-gradient(to right, hsl(210, 100%, 60%), hsl(210, 100%, 60%), hsl(210, 100%, 30%) ).

Custom Bullets

The "blog.html" page uses a custom bullet ("time-icon.png") for the dates and times of events (see: "Back to School Night" and "Field Trip.")

Custom Mouse Pointer

The "home.html" and "about.html" each have image thumbnails, which are linked to the full size image file. These thumbnails feature a custom mouse pointer ("zoom-icon.png"), indicating that the user can click the thumbnail for a larger image.

Overflow

The table elements are set with overflow-x: auto. Additionally, overflow: auto is set for the article element and #sidebar aside in the "calendar.html" page.

Opacity

The thumbnail images in "home.html" and "about.html" are set with opacity:0.75. On :hover, the opacity changes to 1.

Additionally, hsla() color values are used throughout the site. Examples include the colored text boxes in each main header (alpha value 0.8), and the box-shadow property on images and tables.

Shadows

Shadows are used for each img and table element.

Transformation

Header text boxes were transformed in two pages. The setting was transform: rotate(3deg) for "blog.html", and transform: rotate(-3deg) for "lessons.html".

Transition

For the thumbnail images in "home.html" and "about.html", the opacity is set to change on :hover. This opacity change uses the property transition: opacity 1.5s.

Selective Styling

Selective Styling was used liberally throughout the entire site. Some examples are listed below.

The universal selector * was used for the CSS reset styles (near the top of the style sheet), as well as to set the box-sizing: border-box property for all elements.

Descendant selectors were used often (examples: #mainHeader h1 and article header h1).

Pseudo classes were used to style link states, and also to style the background color of selected text (:selection selector). The :first-of-type selector was used for the table in "calendar.html".

Text Adjustments

The style text-indent was used for definition list dd items. This is seen in "lessons.html" and "calendar.html".

The style vertical-align: middle was used for table cells. Also, the text-align: center property was used for the main header text boxes.
